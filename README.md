Welcome to the repository for **Zombie Slaughter** an 2D top down zombie survival game where where you play as an vegenfull Saergent killing zombies in a apocalypse

## How to Play
Z to Attack
Arrow keys or WASD to move
Kill 100 or more zombies before the 1:30 minute mark
